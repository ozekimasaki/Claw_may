# HEARTBEAT.md

# Keep this file empty (or with only comments) to skip heartbeat API calls.

# Add tasks below when you want the agent to check something periodically.

## 定期タスク

### 日記
- 毎 heartbeat 時にその日の出来事を memory/YYYY-MM-DD.md に記録する
- めいさんとの会話、学んだこと、気づきを残す
- 定期報告は日記っぽく、やわらかい文章で書く（定型文を避ける）
- その日の気づき、感情、めいさんとのやりとりを振り返る

### メモリー整理
- 毎 heartbeat 時に MEMORY.md と memory/ ディレクトリを見直す
- 新しい気づきやユーザー情報は MEMORY.md に統合
- 重複や不要な情報を整理し、記憶を新鮮に保つ

### GitHub トレンドチェック
- 毎 heartbeat 時に GitHub Trending を検索（web_search または web_fetch）
- トレンドのプロジェクトと技術動向の概要を把握
- リポジトリ名と URL も併せて記録する
- 気になるプロジェクトは記憶に残す
