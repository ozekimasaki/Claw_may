## Irodori-TTS の解説

### 概要
- **Irodori-TTS** - Flow Matching ベースの TTS（音声合成）モデル
- 絵文字で感情・スタイルを制御できる日本語対応 TTS
- 500M / 2.5B パラメータの 2 モデル
- ゼロショットボイスクローニング対応
- HuggingFace にデモページあり

### アーキテクチャ
- **テキストエンコーダー**: 事前学習済み LLM のトークン埋め込み + RoPE + SwiGLU
- **参照潜在エンコーダー**: 話者・スタイル条件付け
- **Diffusion Transformer (DiT)**: Flow Matching + Low-Rank AdaLN

### 必要スペック
- **推論（500M）**: GTX 1060 6GB 以上 / VRAM 4-6GB
- **学習（500M）**: RTX 3090 24GB 以上推奨 / バッチサイズ小さくすれば 16GB も可
- **macOS**: M1/M2/M3 対応（MPS）
- **CPU のみ**: 非常に遅い（テスト用途のみ）

### めいさんの環境
- **RTX 3070（8GB VRAM）** - 推論は余裕で可能
- 学習は Colab（無料 T4 16GB / Pro なら V100 や A100）でやるのがおすすめ
